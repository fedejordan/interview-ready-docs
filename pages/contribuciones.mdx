# Contribuciones

Esta página esta hecha con Nextra y es [open-source](https://github.com/conanbatt/interview-ready-docs).

Queremos contribuciones de la comunidad para agregar material, hacer correcciones y mejorarla. Para hacerlo simplemente propone un pull request del repo.

## Getting Started

Este proyecto es [Nextra](https://nextra.site/) con pocas modificaciones.

Debería alcanzar con
```
npm install
npm run dev
```
